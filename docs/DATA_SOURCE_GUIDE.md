## Overview

Your Streamlit UI displays data from **3 primary sources**:
1. **Redis** - Fast, real-time data (24h TTL)
2. **PostgreSQL** - Permanent storage (forever)
3. **AWS S3** - PDF file storage (not directly displayed in UI)
